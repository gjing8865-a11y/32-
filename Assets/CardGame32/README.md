# 32 Card Mobile Prototype

This folder contains the first Unity prototype for the 32-card multiplayer table game.

- `Runtime/CardDefinitions.cs` defines the 32-card deck, shuffle, and first-pass hand evaluation.
- `Runtime/CardGamePrototypeController.cs` builds a casual-cartoon mobile table at runtime.
- `Editor/CardGameSceneBuilder.cs` creates `Assets/CardGame32/Scenes/CasualMobilePrototype.unity`.
- `References/casual-cartoon-mobile-reference.png` is the selected visual direction.

Open Unity and use `CardGame32 > Build Casual Mobile Prototype Scene` if the scene has not been generated yet.
